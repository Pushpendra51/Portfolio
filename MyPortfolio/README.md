# Pushpendra — Personal Portfolio

> A fully responsive, single-page developer portfolio built with pure HTML, CSS, and vanilla JavaScript — no frameworks, no dependencies. Optimized for performance and professional aesthetic.

🌐 **Live Site:** [https://Pushpendra51.github.io/Personal-Portfolio](https://Pushpendra51.github.io/Personal-Portfolio)

---

## Features

- **Dark / Light theme** with localStorage persistence
- **Professional Layout** — Clean, modern design with glassmorphism and subtle scroll effects.
- **Project showcase** — Filterable cards highlighting ReelCraft AI, CityZen, and Boat Sales Analytics.
- **Skills section** — Categorized technical expertise and soft skills.
- **Certifications** — Horizontal scroll carousel showcasing professional achievements.
- **Coding profiles** — Integrated links to LeetCode, Codolio, and GitHub.
- **Contact form** — mailto compose functionality with direct email integration.
- **Performance Optimized** — Lazy loading images and GPU-accelerated transitions.
- **Back-to-top** button and scroll progress bar for enhanced navigation.

## Tech Stack

| Layer | Tech |
|-------|------|
| Markup | HTML5 |
| Styling | CSS3 (custom properties, grid, flexbox) |
| Logic | Vanilla JavaScript (ES6+) |
| Hosting | GitHub Pages |

## Running Locally

```bash
# Clone the repo
git clone https://github.com/Pushpendra51/Personal-Portfolio.git
cd Personal-Portfolio

# Serve with any static server, e.g.
python -m http.server 8000
# then open http://localhost:8000
```
